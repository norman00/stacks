# Release notes for HPCRS

## Release `v0.1.0` :

Initial release of the High Performance Compute Reference Stack (HPCRS). This
release features the `sysstacks/hpcrs-clearlinux` Docker image available on
DockerHub*.

- Components included:
  - Clear Linux* OS
  - PyTorch*
  - Horovod*
  - OpenMP
  - Intel® MPI Library
  - Intel® C++ Compiler
  - Intel® Fortran Compiler 
  - Intel® Math Kernel Library

- Versions:
  - Clear Linux* OS -> validated on `33060`
  - PyTorch ->  `1.4.0`
  - Horovod ->  `0.19.0`
  - OpenMP -> `4.5`
  - Intel® MPI Library ->  `2019.7.217`
  - Intel® C++ Compiler ->  `parallel_studio_xe_2020_update1_composer_edition`
  - Intel® Fortran Compiler ->  `parallel_studio_xe_2020_update1_composer_edition`
  - Intel® Math Kernel Library ->  `2020.1.217`
